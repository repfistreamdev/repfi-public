## What is RepFi?

RepFi is a **fitness bounty platform**. Athletes go live, complete verified tasks (starting with **push-ups**), and earn **$0.01 per verified rep**. Payouts are funded by a transparent **donation pool** — community contributions plus **$RepFi** token trading fees on Solana (pump.fun).

No gym membership. No opaque middlemen. Reps are counted by **AI pose tracking** in the browser — not self-reported.

## How it works

| Step | What happens |
|------|----------------|
| 1. **Register** | Gmail + nickname + Solana address (paste only — no wallet extension) |
| 2. **Join a room** | 10 live rooms; one athlete per room |
| 3. **Go live** | Camera + microphone; viewers watch and chat |
| 4. **AI verifies** | MediaPipe Pose tracks your skeleton; only correct push-ups count |
| 5. **Earn** | +$0.01 per verified rep, live balance |
| 6. **Withdraw** | Minimum $1.00 → sent to your Solana address |

## Why this is not a scam

- **Free to earn** — no payment required to register or participate
- **AI-verified reps** — not honor system, not button clicking
- **Transparent pool** — live balance at [repfi.stream/pool](https://repfi.stream/pool)
- **Your wallet** — we never ask for seed phrases or private keys
- **No guaranteed returns** — pay scales with verified effort and pool liquidity

**Full breakdown:** [docs/WHY_NOT_A_SCAM.md](docs/WHY_NOT_A_SCAM.md)
